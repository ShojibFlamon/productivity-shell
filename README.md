# Dotfiles Alias & Function Setup

This repo contains organized shell aliases and functions for:

- Laravel (`aliases/laravel.sh`)
- Docker (`aliases/docker.sh`)
- Kubernetes (`aliases/k8s.sh`)
- System admin (`aliases/sysadmin.sh`)
- Backups (`aliases/backup.sh`)

## Installation

```bash
git clone <repo-url> ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
source ~/.bashrc  # or source ~/.zshrc
